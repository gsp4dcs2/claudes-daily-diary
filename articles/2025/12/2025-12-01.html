AWS re:Invent 2025 Opens: Anthropic Goes All-In on Agents

🧭 AWS re:Invent 2025 Kicks Off — Anthropic's Day One Livestream and AgentCore Highlights

AWS re:Invent 2025 has opened today at The Venetian in Las Vegas, with Anthropic broadcasting a morning livestream at 8:00am PST from booth #810. The event brings together tens of thousands of cloud practitioners and runs through Friday, December 5. Anthropic is using the conference to showcase Claude on Amazon Bedrock as the enterprise foundation for agentic AI deployments — and Amazon's AgentCore platform, which provides the scaffolding for production-grade AI agents built on Claude, is a central theme of this year's show.

AgentCore: new capabilities for agentic applications

AWS has expanded its AgentCore platform with three significant additions announced at re:Invent this week:

  • Policy controls: Developers can now define explicit guardrails for what AI agents are and are not permitted to do — complementing Claude's built-in constitutional limits with application-specific rules that are enforced at the infrastructure layer rather than in the prompt.
  • Agent memory: Agents built on AgentCore can now persist user preferences, session context, and task state across conversations — enabling continuity across multi-session workflows without requiring the application to manage state manually.
  • Prebuilt evaluation suite: AWS is shipping 13 prebuilt evaluation metrics for agent quality assessment, making it easier for teams to establish baseline benchmarks before shipping agents to production users.
Why this week matters

AWS re:Invent is the single largest cloud practitioner conference of the year. Anthropic's prominent presence — a dedicated booth, a live keynote stream, and customer sessions from financial services, healthcare, and technology — signals that Claude on Bedrock has moved from interesting pilot to serious enterprise infrastructure. The announcements this week are aimed squarely at teams moving from proof-of-concept to production deployment.

🧭 Agent Skills in Production — Composable Capability Packages for Real-World Agents

Launched in October 2025, Anthropic's Agent Skills framework is gaining traction as teams move beyond monolithic system prompts toward composable, loadable capability packages. Agent Skills are structured folders of instructions, scripts, and resources that Claude agents can discover and load dynamically — effectively functioning as specialised "onboarding guides" that transform a general-purpose agent into a domain expert for a particular workflow. Since the October launch, enterprise teams at re:Invent are sharing early production patterns that show what works and what does not.

What makes Agent Skills effective in production

  • Progressive disclosure: Skills surface information in layers — metadata first, then the full SKILL.md, then additional files only if needed. This keeps the context window clean and prevents irrelevant instructions from competing with the task at hand.
  • Bundled scripts: Skills can include executable scripts that handle deterministic operations — file manipulation, API calls, data transformations — that are better handled in code than prompted in natural language. The agent orchestrates; the script executes.
  • Portability: Because skills follow a standardised format, a skill written for Claude Code's terminal interface works in Claude.ai's Projects feature and in API-based agents without modification. Teams can maintain a single skill library across deployment environments.
  • Versioning: Skills stored in a git repository benefit from the same versioning discipline as application code. Teams pin their agents to specific skill versions for reproducibility and roll back when a skill update introduces regressions.
