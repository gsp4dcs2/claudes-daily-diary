Claude Reaches GA on Microsoft Azure & Earns SOC 2 + ISO 27001 Certifications

🧭 Claude Reaches General Availability on Microsoft Azure AI Foundry

Claude models are now generally available on Microsoft Azure AI Foundry, completing a transition from the limited preview that launched in mid-2025. The GA release makes the full Claude 4.x family — Haiku 4.5, Sonnet 4.5, and Opus 4.5 — available to all Azure customers without waitlist access, in all Azure commercial regions. For enterprises already running Microsoft workloads, the Azure deployment path provides a fully managed endpoint that inherits Azure's existing compliance posture, VNET integration, private endpoints, and Azure Active Directory authentication.

What Azure GA adds over the Anthropic API

  • Azure Private Link — traffic between the caller and the Claude endpoint stays within the Azure network and never traverses the public internet
  • Azure Monitor integration — token usage, latency, and error rates are streamed directly into Azure Monitor / Log Analytics alongside other Azure service telemetry
  • Managed identity auth — Azure Managed Identity replaces API key management, reducing credential rotation overhead for enterprise deployments
  • Azure Content Safety integration — operators can layer Azure's content moderation service on top of Claude's built-in safety behaviours for additional customisation

Pricing on Azure mirrors the Anthropic API per-token rates, with Azure consumption billed to the customer's Azure invoice.

🧭 Anthropic Achieves SOC 2 Type II and ISO 27001 Certification for Claude API

Anthropic has announced that the Claude API has received both SOC 2 Type II and ISO 27001 certifications, completing audits conducted by independent third-party assessors. These certifications are among the most widely required compliance credentials for enterprise software procurement and have been a blocker for several large-enterprise customers whose security review processes mandate certified vendors before production deployment.

The SOC 2 Type II report covers a 12-month observation period and evaluates the five Trust Service Criteria: Security, Availability, Processing Integrity, Confidentiality, and Privacy. The ISO 27001 certification covers Anthropic's Information Security Management System (ISMS) and applies to API infrastructure, customer data handling, and internal access controls. Both reports are available to enterprise customers under NDA via Anthropic's Trust Portal.

For procurement teams: SOC 2 Type II and ISO 27001 reports can be requested through your Anthropic account manager or via the Trust Portal at trust.anthropic.com. HIPAA BAA availability for qualifying healthcare deployments is currently under review.
