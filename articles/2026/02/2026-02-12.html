Self-Serve Enterprise Plans Launch & MCP Passes 10,000 Servers

✦ Self-Serve Enterprise Plans — No Sales Call Required

Anthropic has opened Enterprise plan sign-ups directly on its website, eliminating the requirement for a sales conversation. Any organisation can now purchase an Enterprise plan, provision a workspace, configure SSO, and begin inviting team members in minutes. The self-serve tier bundles access to Claude, Claude Code, and Cowork under a single seat type.

Organisations with more complex requirements — HIPAA readiness, usage-commitment tiering, custom data retention policies, or bespoke SLAs — can still engage the sales team. But for mid-market companies that previously waited days or weeks for Enterprise access while a sales process played out, the default path is now fully automated. Anthropic has indicated this is part of a broader effort to compress the enterprise adoption cycle as the competitive environment for AI tools intensifies.

What's in Enterprise vs. Teams: Enterprise adds SSO/SAML, audit logs, workspace usage analytics, custom data retention settings, HIPAA eligibility, priority support, and access to the Enterprise Analytics API. If your team is on Teams today and needs any of these, the self-serve upgrade path is now a button click.

✦ MCP Directory Crosses 10,000 Active Public Servers

The Model Context Protocol public server directory has crossed 10,000 active listings this week — a significant milestone for an ecosystem that launched its formal directory in mid-2025. The registry covers every major category: developer toolchains (GitHub, Sentry, Linear, Jira), productivity suites (Google Drive, Notion, Asana, Slack), data platforms (Snowflake, BigQuery, Postgres), scientific databases, financial data providers, and government open-data APIs.

Anthropic's developer-relations team attributes the acceleration to three factors: the Tool Search Tool reaching GA (making large connector libraries economically viable), the Claude Code Plugin Marketplace (which lowered the publishing barrier), and the community's appetite to make local tools accessible to agent workflows without full-stack integration effort.

Building your own MCP server: If you have an internal data source, API, or tool that your team uses regularly, building an MCP server and pointing Claude Code at it is now a well-documented pattern. Anthropic's MCP SDK supports TypeScript and Python, and the directory submission process takes under 30 minutes for a working server.
